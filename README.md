# Run code

Execute `run.sh`. If you want more consumers, modify that file e.g:

- From 1 (one) consumer
    ```shell
    docker-compose up -d --scale consumer=1
    ```

- To 5 (five) consumers
    ```shell
    docker-compose up -d --scale consumer=5
    ```
